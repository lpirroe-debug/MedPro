🔧 Configuración de Google API
Pasos para obtener el Client ID (una sola vez, ~10 minutos):
1. Ve a console.cloud.google.com
2. Crea un nuevo proyecto (ej: "MedPro")
3. Ve a APIs y servicios → Biblioteca → activa Google Drive API
4. Ve a APIs y servicios → Credenciales → Crear credenciales → ID de cliente OAuth
5. Tipo de aplicación: Aplicación web
6. En "Orígenes de JavaScript autorizados" agrega la URL donde usas el app (ej: http://localhost o tu dominio)
7. Copia el Client ID generado y pégalo aquí abajo
Formato: números-letras.apps.googleusercontent.com
